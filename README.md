# Walkr

> **WIP**  
> ![Coming Soon](https://img.shields.io/badge/status-coming--soon-orange)

Code-first product demo tool. Script website interactions as animated cursor walkthroughs with a Remotion-style TypeScript API and timeline editor.

## Packages

| Package | Description |
| --- | --- |
| `@walkr/core` | Core scripting API for describing walkthroughs in TypeScript. |
| `@walkr/engine` | Playback engine for rendering and controlling walkthrough timelines. |
| `@walkr/studio` | React-based timeline editor application for building demos visually. |
| `@walkr/cli` | CLI for building, running, and managing Walkr demos. |
| `@walkr/playwright` | Headless capture adapter powered by Playwright tooling. |
